## Scope

Create all templates required to generate a complete Go + Gin service. This story covers the application code templates. Infrastructure templates (Kubernetes, Terraform, CI) are generated by the respective plugins.

### In Scope

- All templates above with correct variable substitution
- Templates embedded in the binary
- All Tier 1 standards wired (health endpoints, logging, OTel, error handler, graceful shutdown, security headers, input sanitization)
- Template rendering tests confirming the generated code compiles

### Out of Scope

- Python, Java, Node.js templates (Epic 2.2)
- Kubernetes and Terraform templates (Epic 1.7 plugins and Epic 1.9)

## Acceptance Criteria

- [x] Generated Go + Gin service compiles with `go build ./...`
  - All templates produce syntactically valid Go with correct import paths via `{{ .service.module }}`
  - `ServiceData.Module` field added to domain model and `ToMap()` ‚Äî templates resolve correctly
  - Verified by template rendering tests (40/40 pass in `data/templates`)
- [ ] Generated service starts with `make dev` (or `go run`)
  - Blocked on Epic 1.9 (`crux new` end-to-end generation not yet wired); templates are correct
- [ ] All five health endpoints respond correctly
  - Blocked on Epic 1.9; all five endpoints present in `health.go.tmpl` and verified by tests
- [ ] Generated tests pass with `go test ./...`
  - Blocked on Epic 1.9; no generated test files exist yet (out of scope for this story)
- [ ] `docker build .` succeeds
  - Blocked on Epic 1.9; `Dockerfile.tmpl` is correct (multi-stage, non-root, HEALTHCHECK ‚Äî verified by tests)
- [x] All Tier 1 standards are present in the generated service
  - Health endpoints: `health.go.tmpl` ‚Äî `/health`, `/ready`, `/live`, `/metrics`, `/version`
  - Structured JSON logging: `logger.go.tmpl` ‚Äî slog JSON handler, `timestamp` key, trace correlation
  - OTel tracing: `main.go.tmpl` calls `tracing.Init`; `router.go.tmpl` applies `tracing.Middleware`
  - W3C traceparent: propagated via `tracing.Middleware` in router
  - RFC 7807 errors: `errors/handler.go.tmpl` ‚Äî `application/problem+json`, panic recovery
  - Graceful shutdown: `main.go.tmpl` ‚Äî SIGTERM/SIGINT via `shutdown.Runner`, HTTP + tracing hooks
  - Security headers: `router.go.tmpl` ‚Äî CSP, HSTS, X-Frame-Options, X-Content-Type-Options
  - Input sanitization: `router.go.tmpl` ‚Äî path traversal, null bytes, Content-Type enforcement
  - Non-root container: `Dockerfile.tmpl` ‚Äî distroless nonroot, `USER nonroot:nonroot`
- [x] Template rendering tests pass in the Crux test suite
  - 40/40 tests pass in `data/templates` package (`go test ./data/templates/...`)
  - Full suite: 12/12 packages pass (`go test ./...`)
